  • Let me in app

    It is a simple and useful app to store simple information such as web site ID / password, bank, card which is easy to forget. Lightweight and easy to use, the intuitive UI makes it easy to store and view at any time. All data is encrypted and stored only on the device, not sent anywhere and can be used even without a network connection. If necessary, you can back up and restore from 'Export Data' / 'Import Data' menu. * Made with React Native.

    March, 2019
